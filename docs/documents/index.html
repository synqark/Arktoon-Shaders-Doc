説明に使うモデルについて

全編通して、販売3Dモデルである量産型のらきゃっとを使用して説明いたします。
量産型のらきゃっと は、バーチャル美少女のらきゃっと(のらきゃっとチャンネル) から
公開・販売されている、文字通りのらきゃっとちゃんのマスプロダクションモデルです。
